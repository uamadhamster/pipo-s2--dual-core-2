# Инструкция по прошивке устройства стандартной программой Rockchip Batch Tool #

**1. Требования**: RRockchip Batch Tool v1.5 Файл прошивки PIPO S2, Персональный компьютер или ноутбук /нетбук под управлением операционной системы Windows (для ПК желательно наличие ИБП) USB-кабель.

**2. Первоначальная настройка:** Извлечь RRockchip Batch Tool v1.5 (далее RBT) из архива. ПО включает в себя необходимые драйверы для устройства. Распакуйте скачанный образ прошивки Запустите RBT и выберите файл прошивки (…). Если все сделано правильно, вы увидите информацию в окне программы - текущую версию прошивки, версию Android, ревизию платы (RK30) и т.д.

**3. Установка драйверов устройств:** Выключите устройство Подключите к компьютеру. Нажмите кнопку громкости +, затем подключите USB кабель к компьютеру и планшету(кнопка громкости остается нажатой). Когда система увидит новое устройство, отпустите кнопку. При запросе установки драйвера для устройства, перейдите в каталог RBT и выберите папку "Drivers" и операционная система установит необходимый драйвер. После завершения установки драйвера для устройства одно из полей RBT должно сменить цвет с серого на зеленый.

**4. Установка новой прошивки:** После подключения планшета, нажмите кнопку "Upgrade" Вы должны увидеть в зеленом поле оранжевый текст, описывающий каждый этап обновления устройства. Если все сделано правильно, вы увидите такое сообщение:

**Upgrade Done Success<%d> Fail<%d> Time<%d>ms****Of course the %d parts replaced with numbers**



# Заключение #

Если прошивка прошла в штатном режиме, планшет начнет загружаться. Если этого не произошло, то просто нажмите на кнопку питания, и ждите загрузки.

Дополнение: Если нажать не "Upgrade", а "Restore", то происходит еще дополнительное внутреннее форматирование и далее прошивка также встает нормально.
